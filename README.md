# AutoTune — Конфигурация для учета детейлинга и ТО на платформе 1С 8.3

Учебный проект нестандартной конфигурации, демонстрирующий владение базовыми механизмами платформы «1С:Предприятие 8.3».

## 🛠 Технологический стек & Объекты
* **Справочники:** Клиенты, Автомобили (с объектной связью), Номенклатура, Услуги.
* **Документы:** `ПоступлениеТоваров` (складской приход), `ЗаказНаряд` (оказание услуг и списание материалов).
* **Регистры накопления:** `ОстаткиМатериалов` (тип: Остатки) и `ВыручкаМастерской` (тип: Обороты).
* **Бизнес-логика:** Написаны процедуры проведения документов в транзакции (Модули объектов).
* **UX/UI:** Реализован асинхронный клиент-серверный триггер (`&НаСервереБезКонтекста`) для динамической подгрузки цен услуг при оформлении заказа.
* **Отчетность:** Создан аналитический отчет на базе **СКД (Система компоновки данных)** с выводом структуры доходов в виде круговой диаграммы.

## 📂 Структура репозитория
Исходный код выгружен стандартными средствами платформы в текстовые форматы `.bsl` (код модулей) и `.xml` (метаданные) для обеспечения версионирования.
